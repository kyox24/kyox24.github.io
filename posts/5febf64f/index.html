记一次奇怪的Template Render Error

发表于2021-05-09|更新于2021-05-09|备忘记录
|阅读量:

前言

因为暂时无法复现Bug所以报错信息也丢失了，借这个机会我打开了iTerm2的无限回滚。

Bug表现为hexo d -g之后显示Template Render Error。

尝试的解决方法一

把文章中非代码块部分出现的双重以上的大括号用空格分开，即{{ -> { {。

尝试的解决方法二

用到标签外挂的地方，消除大括号和百分号之间出现的空格，即{ % note warning % } -> {% note warning %}

最后就莫名其妙解决了…
