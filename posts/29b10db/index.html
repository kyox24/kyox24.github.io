论文信息

SinGAN: Learning a Generative Model from a Single Natural Image

GitHub源码

概述

本文提出了一种可以从单一图像中学习的GAN 架构，这种架构是非条件的(unconditional)，并且有很高程度的通用性，可以解决任意尺寸的图像生成、图像合成、超高分辨率和从单一图像生成视频等任务。与传统GAN 的明显不同在于神经网络架构和Loss Function 上。

关键点

SinGAN的Pipeline

暂时不讨论具体细节，总的来说SinGAN秉承着从粗到细（coarse-to-fine）的多层结构，首先粗略地掌握图片的全局上的结构，再逐渐抠细节，先看一下SinGAN的Pipeline：

这里的Generator构成了一种金字塔形的结构，为了不混淆，根据我个人的理解对"金字塔形结构"进行进一步的解释。

  • 在SinGAN中定义了一系列Generator{G0,…,GN}\left\{ G_0,\dots,G_N\right\}{G0​,…,GN​}，其中G0G_0G0​ 的特点是输入输出的图片尺寸最大、生成效果最精细、所以也是我们最后需要的，形象上可以认为G0G_0G0​在金字塔顶端
  • GNG_NGN​则与之相对，输入输出尺寸最小、生成效果最粗糙，形象上可以认为GNG_NGN​在金字塔底端
  • 训练过程是从金字塔结构中自底向上的

具体的神经网络架构

由于Generator和Discriminator的架构一样，这里以Generator为例进行说明。

输入输出：

  • 输入：底层（GNG_NGN​）的输入仅为噪声，除了GNG_NGN​外，都遵循上图，输入为它之前一层的一个上采样与一个高斯噪声的相加
  • 输出：一个与输入尺寸相同的图像

论文里没有提到上采样的具体方法，这一部分如果好奇的话可以参见源码

神经网络内部架构：

  • 共5个卷积块
  • 每个卷积块
    • 卷积层，kernal size = 3×33\times33×3，stride = 1， padding = 1
    • BN层
    • 激活函数LeakyReLU
  • 残差

实际上还有个与传统GAN 架构不同的地方在于这个网络架构中没有全连接层

数学表达

我们把通过Generator生成的图像记作x~n(0≤n≤N)\widetilde{x}_n(0 \leq n \leq N)xn​(0≤n≤N)，特别的，对于n=Nn=Nn=N时，GNG_NGN​做的仅仅是把高斯噪声映射到一个生成样本：

x~N=GN(zN)\widetilde{x}_N=G_N(z_N) xN​=GN​(zN​)

对于n>0n\gt 0n>0时：

x~n=Gn(zn, (x~n+1)↑r)\widetilde{x}_n=G_n(z_n, \ (\widetilde{x}_{n+1})\uparrow^r) xn​=Gn​(zn​, (xn+1​)↑r)

其中(x~n+1)↑r(\widetilde{x}_{n+1})\uparrow^r(xn+1​)↑r 表示之前一层的倍率为rrr 的上采样，我们可以把上式写成更为具体的形式：

x~n=(x~n+1)↑r+ψn(zn+(x~n+1)↑r)\widetilde{x}_n=(\widetilde{x}_{n+1})\uparrow^r+\psi_n(z_n+(\widetilde{x}_{n+1})\uparrow^r) xn​=(xn+1​)↑r+ψn​(zn​+(xn+1​)↑r)

其中ψn\psi_nψn​即为上述的有5个卷积块的神经网络。

SinGAN的Loss Function

SinGAN的Loss Function整体上由两项构成，adversarial loss与reconstruction loss：

minGnmaxDnLadv(Gn,Dn)+αLrec(Gn)\mathop{min}\limits_{G_n}\mathop{max}\limits_{D_n}\mathcal{L}_{adv}(G_n, D_n)+\alpha\mathcal{L}_{rec}(G_n) Gn​min​Dn​max​Ladv​(Gn​,Dn​)+αLrec​(Gn​)

  • adversarial loss中用的是WGAN-GP 的loss：

Ladv=Ex~∼Pg[D(x~)]−Ex~∼Pr[D(x)]+λEx^∼Px^[(∣∣∇x^D(x^)∣∣2−1)2]\mathcal{L}_{adv}=\mathop{\mathbb{E} }\limits_{\widetilde{x}\sim\mathbb{P}_g}[D(\widetilde{x})]-\mathop{\mathbb{E}}\limits_{\widetilde{x}\sim\mathbb{P}_r}[D(x)]+\lambda\mathop{\mathbb{E} }\limits_{\hat{x}\sim\mathbb{P}_{\hat{x}}} \left [ (||\nabla_{\hat{x}}D(\hat{x})||_2-1)^2 \right ] Ladv​=x∼Pg​E​[D(x)]−x∼Pr​E​[D(x)]+λx^∼Px^​E​[(∣∣∇x^​D(x^)∣∣2​−1)2]

  • reconstruction loss 是为了保证存在一组噪声可以由之生成原图像，在这里特定选择了这组噪声为{zNrec,zN−1rec,…,z0rec}={z∗,0,…,0}\{z^{rec}_N,z^{rec}_{N-1},\dots,z^{rec}_0\}=\{z^*,0,\dots,0\}{zNrec​,zN−1rec​,…,z0rec​}={z∗,0,…,0}，其中z∗z^*z∗是一组特定的固定的噪声，并把使用这种噪声生成的第nnn个图像记作x~nrec\widetilde{x}^{rec}_nxnrec​，那么有：

Lrec={∣∣GN(z∗)−xN∣∣2if n=N∣∣Gn(0,(x~n+1rec)↑r)−xn∣∣2if n<N\mathcal{L}_{rec}=\left\{ \begin{aligned} & ||G_N(z^*)-x_N||^2 & \text{if}\ n=N \\ & ||G_n(0,(\widetilde{x}^{rec}_{n+1})\uparrow^r)-x_n||^2 & \text{if} \ n \lt N \end{aligned} \right . Lrec​={​∣∣GN​(z∗)−xN​∣∣2∣∣Gn​(0,(xn+1rec​)↑r)−xn​∣∣2​if n=Nif n<N​

实际上reconstruction loss 还用来决定模型所用噪声的标准差

回顾SinGAN的Pipeline

这里主要是初看论文时对于Effective Patch Size，也就是下图红框部分有些不理解，记录一下我个人的理解：

论文中这张配图下方写到：

the effective patch size decreases as we go up the pyramid (marked in yellow on the original image for illustration).

先说结论这里虽然提到patch size 会随着自底向上的过程中不断变小，但是我认为这里说的变小并不是绝对意义上的变小，而是patch size相对于整个生成出来的图片在变小。

接下来是我这么认为的原因，首先需要明确在这里的patch size 可以理解为receptive field的大小，这当然也是论文中写到的：

The architecture of DnD_nDn​ is the same as the net ψn\psi_nψn​ within GnG_nGn​, so that its patch size (the net’s receptive field) is 11 × 11.

可以注意到上面提到这个patch size 应该是固定为11×1111\times 1111×11的，这通过神经网络架构中kernal size、stride和padding 就可以计算得知，具体的计算方法可以参见彻底搞懂感受野的含义与计算，写的非常好！

所以显然patch size并不会发生绝对的变化，而是相对的。
