带有正则化经验误差最小化(Regularized Loss Minimization, RLM)

这个问题中目的函数的形式：

J(w)=∑i=1nℓi(w)+λr(w)J(\bold w)=\sum_{i=1}^{n}\ell_i(\bold w)+\lambda r(\bold w) J(w)=i=1∑n​ℓi​(w)+λr(w)

其中第一项被称为损失项，第二项被称为正则项。

  • 二乘损失

ℓi(w)=(wTxi−yi)\ell_i(\bold w)=(\bold w ^T \bold x_i - y_i) ℓi​(w)=(wTxi​−yi​)

  • 绝对损失

ℓi(w)=∣wTxi−yi∣\ell_i(\bold w)=\left |\bold w ^T \bold x_i - y_i \right| ℓi​(w)=∣∣​wTxi​−yi​∣∣​

  • ε\varepsilonε-不敏感损失

ℓi(w)=max(−(wTxi−yi)−ε, 0, (wTxi−yi)−ε)\ell_i(\bold w)= \mathop {max}(-(\bold w^T\bold x_i-y_i)-\varepsilon,\ 0, \ (\bold w^T\bold x_i-y_i)-\varepsilon) ℓi​(w)=max(−(wTxi​−yi​)−ε, 0, (wTxi​−yi​)−ε)

二分类问题

yi∈{+1, −1}y_i \in \{ +1, \ -1\}yi​∈{+1, −1}

  • 判别函数

y^i={+1wTxi>0−1wTxi≤0\hat{y}_i=\left\{ \begin{aligned} +1& \quad \bold w^T\bold x_i \gt0 \\ -1 & \quad \bold w^T\bold x_i \leq0 \end{aligned} \right . y^​i​={+1−1​wTxi​>0wTxi​≤0​

  • 合页损失函数（Hinge Loss Function）

ℓi(w)=max(0, 1−yiwTxi)\ell_i(\bold w)=\mathop{max}(0,\ 1-y_i\bold w^T\bold x_i) ℓi​(w)=max(0, 1−yi​wTxi​)

  • 逻辑损失函数（Logistics Loss Function）

ℓi(w)=log(1+exp(−yiwTxi))\ell_i(\bold w)=\mathop{log}\left(1+exp(-y_i\bold w^T\bold x_i)\right) ℓi​(w)=log(1+exp(−yi​wTxi​))

正则项

常见的正则项如下：

  • L2L_2L2​正则项

r(w)=12∥w∥22r(\bold w) = \dfrac{1}{2}\left \lVert \bold w \right \rVert_2^2 r(w)=21​∥w∥22​

  • L1L_1L1​正则项

r(w)=∥w∥1r(\bold w) = \left \lVert \bold w \right \rVert_1 r(w)=∥w∥1​

  • L0L_0L0​正则项

r(w)=∥w∥0r(\bold w) = \left \lVert \bold w \right \rVert_0 r(w)=∥w∥0​

  • LpL_pLp​正则项

r(w)=1p∥w∥ppr(\bold w) = \dfrac{1}{p} \left \lVert \bold w \right \rVert_p^p r(w)=p1​∥w∥pp​

  • L∞L_\inftyL∞​正则项

r(w)=∥w∥∞r(\bold w) = \left \lVert \bold w \right \rVert_\infty r(w)=∥w∥∞​

  • 弹性网络正则项（ElasticNet）

r(w)=p⋅12∥w∥22+(1−p)∥w∥1r(\bold w) = p \cdot \dfrac{1}{2} \left \lVert \bold w \right \rVert_2^2+(1-p)\left \lVert \bold w \right \rVert_1 r(w)=p⋅21​∥w∥22​+(1−p)∥w∥1​

分类问题的指标

这里同样是按照课件的写法，网上有更好的参考，如一文看懂机器学习指标

  • 正确率（Accuracy）

#{i∣y^i=yi}n\dfrac{\#\{i|\hat{y}_i=y_i\} }{n} n#{i∣y^​i​=yi​}​

  • 精确率（Precision）

#{i∣y^i=+1, yi=+1}#{i∣y^i=+1}\dfrac{\#\{i|\hat{y}_i=+1, \ y_i=+1\} }{\# \{ i|\hat{y}_i=+1\} } #{i∣y^​i​=+1}#{i∣y^​i​=+1, yi​=+1}​

  • 召回率（Recall)

注意分母与精确率的分母不相同

#{i∣y^i=+1, yi=+1}#{i∣yi=+1}\dfrac{\#\{i|\hat{y}_i=+1, \ y_i=+1\} }{\# \{ i|y_i=+1\} } #{i∣yi​=+1}#{i∣y^​i​=+1, yi​=+1}​

  • ROC曲线（Receiver Operating Characteristic curve）

横轴:#{i∣y^i=+1, yi=+1}#{i∣yi=+1}\text{横轴}:\dfrac{\#\{i|\hat{y}_i=+1, \ y_i=+1\} }{\# \{ i|y_i=+1\} } 横轴:#{i∣yi​=+1}#{i∣y^​i​=+1, yi​=+1}​

纵轴:#{i∣y^i=+1, yi=+1}#{i∣yi=−1}\text{纵轴}:\dfrac{\#\{i|\hat{y}_i=+1, \ y_i=+1\} }{\# \{ i|y_i=-1\} } 纵轴:#{i∣yi​=−1}#{i∣y^​i​=+1, yi​=+1}​

  • PR曲线（Precision-Recall curve）

    横轴召回率，纵轴精确率。
